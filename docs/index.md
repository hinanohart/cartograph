# Cartograph

Cartograph diagnoses foundation models on Félix Guattari's four schizoanalytic
functors: **F** (Flows), **Φ** (Machinic Phyla), **T** (Existential
Territories), and **U** (Universes of Reference).

This is research software. The Phase 1a alpha-pre release ships **Φ** and **U**
as production-quality functors; **F** and **T** are typed stubs that land in
Phase 1b. See [decisions/ADR-0001-hybrid-s.md](decisions/ADR-0001-hybrid-s.md)
for the architecture rationale.

## Status

- Phase 1a (alpha-pre): v0.1.0a0 — Φ + U + adapter Protocol
- Phase 1b (alpha): v0.2.0 — F + T integrated report
- Phase 1c (alpha + preprint): v0.3.0 — arXiv submission

## What this is not

- Not a turnkey product. Construct validity of T is gated on external peer
  review (W3 mitigation, ADR-0001).
- Not "complete" or "permanent": this is a 5-6 month plan that may revise
  scope when Phase 1b construct-validity peers report back.
